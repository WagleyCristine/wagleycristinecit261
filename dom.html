DOM- Create Element, Append Child, Insert Before, and Remove Child.

This example uses a list to show how we can create new elements,
append or (add the new elements to the bottom of the list), insert elements
before a specified line item, or remove a child element from the DOM or
(Document Oriented Model).
Pink Floyd is a the top of our list using insertBefore
Modest Mouse has been added to the bottom of the list using appendChild.
Both band names are elements that have been created. A new element was
also added to the list with append and removed from the list with removeChild.
This is noted in the code to explain where this took place.

  1. insertBefore()

  2. createElement()

  3. appendChild()

  4. removeChild()
