Local Storage

Local Storage is used in key,value pairs and stores data locally in the browser settings.
This information stays even after a window is closed and has to be manually removed.

Storing Data


In this first example the user can enter whatever they like and once the choose to save the information it will me entered into the localStorage.
This stores the information until it is manually removed, even if the window is closed out and reopened. To view any of this
information in local storage click F12 and use the developer tools to view local storage. The remove button on the first example removes one specific item.

Retrieving Data

In this example information has been saved into the local storage and can be loaded for the user
to view after clicking the save button and then the load button. Notice if you try to load first the information
will not come up. This is set into code as an object and upon clicking the save button it will save that object
to your local storage.
